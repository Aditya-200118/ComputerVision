\section{Introduction}
This report details the implementation of a multi-scale Laplacian-of-Gaussian (LoG) edge detection algorithm, commonly referred to as Edge Focusing. By tracking zero-crossings from a coarse scale ($\sigma = 5.0$) down to a fine scale ($\sigma = 1.0$), the pipeline mathematically combines the noise-immunity of large convolution kernels with the precise spatial localization of small kernels. The performance is objectively verified against a single-scale LoG approach using Pratt's Figure of Merit (PFOM).

\section{Algorithm and Methodology}

\subsection{Scale-Normalized LoG Kernel}
To ensure the convolution response does not mathematically decay as the kernel size decreases, the pipeline implements Lindeberg's Scale-Space Amplitude Normalization. By altering the denominator from $\sigma^4$ to $\sigma^2$, the peak response remains consistent across all scales. The kernel is evaluated over the strict range $[-4\sigma, +4\sigma]$ to guarantee an odd-sized convolution mask:
\begin{equation}
    \text{LoG}(x, y) = -\frac{1}{\pi \sigma^2} \left[ 1 - \frac{x^2 + y^2}{2\sigma^2} \right] \exp\left(-\frac{x^2 + y^2}{2\sigma^2}\right)
\end{equation}
To ensure computational efficiency at large scales (e.g., $\sigma = 5.0$), the 2D convolution is performed in the frequency domain using the Fast Fourier Transform (FFT).

\subsection{Multi-Scale Edge Focusing (Tracking)}
The Edge Focusing algorithm dictates that true structural edges exist at coarse scales, while noise does not. A base threshold (anchored at 5\% of the maximum LoG response) is modulated by a spatial bimodal penalty map to isolate legitimate zero-crossings. 

Once zero-crossings are detected at scale $\sigma_i$, morphological dilation (2 iterations) is applied to create a boolean tracking mask. When processing the next scale ($\sigma_{i-0.5}$), the algorithm is strictly constrained to search for zero-crossings only within this active mask.

\begin{algorithm}[H]
\DontPrintSemicolon
\SetKwInOut{Input}{input}
\SetKwInOut{Output}{output}

\Input{Normalized Grayscale Image $I$, $\sigma_{start}=5.0$, $\sigma_{end}=1.0$, step $\Delta\sigma=0.5$}
\Output{Final focused edge map $E$, intermediate maps for all $\sigma$}

Initialize $SearchMask \leftarrow \text{None}$\;
Initialize dictionary $SavedMaps$\;

\For{$\sigma \leftarrow \sigma_{start}$ \KwTo $\sigma_{end}$ \textbf{step} $-\Delta\sigma$}{
    $K \leftarrow \text{GenerateScaleNormalizedLoG}(\sigma, \text{range}=[-4\sigma, 4\sigma])$\;
    $L \leftarrow \text{Convolve2D\_FFT}(I, K)$\;
    
    $L_{norm} \leftarrow L / \max(|L|)$\;
    $T_{map} \leftarrow \text{CalculateAdaptiveThresholdMap}(I)$\;
    
    $E_{\sigma} \leftarrow \text{FindZeroCrossings}(L_{norm}, SearchMask, T_{map})$\;
    
    \If{$\sigma \in \{5.0, 4.0, 3.0, 2.0, 1.0\}$}{
        $SavedMaps[\sigma] \leftarrow E_{\sigma}$\;
    }
    
    $SearchMask \leftarrow \text{Dilate}(E_{\sigma}, \text{iterations}=2)$ \tcp*{Update tracking window}
}

\Return $SavedMaps$\;
\caption{Multi-Scale Edge Focusing Pipeline}
\end{algorithm}

\section{Experimental Verification Framework}
To mathematically verify that Edge Focusing is superior to a Single-Scale LoG operator, the final output must be evaluated against a proxy ground truth (OpenCV Canny optimized via Otsu thresholding). \textbf{Pratt's Figure of Merit (PFOM)} is calculated using Euclidean distance transforms. A PFOM score closer to $1.0$ indicates perfect overlap, while lower scores mathematically penalize excessive noise or edge drift.

\section{Conclusion and Analysis}
Based on the experimental verification, the Multi-Scale LoG operator (Edge Focusing) is demonstrably superior to a Single-Scale LoG operator. 

As evidenced by the visual outputs and mathematically proven by Pratt's Figure of Merit (PFOM), applying a single-scale LoG at $\sigma=1.0$ results in excellent spatial localization but extreme susceptibility to high-frequency noise. Conversely, applying a single-scale LoG at $\sigma=5.0$ eliminates noise but causes structural edges to drift significantly from their true locations. 

The Edge Focusing pipeline successfully bridges this gap. By utilizing a coarse-to-fine morphological tracking mask, the algorithm inherits the noise-immunity of the $\sigma=5.0$ scale, while retaining the precise pixel localization of the $\sigma=1.0$ scale. Across all three test images, the multi-scale approach yielded the highest structural fidelity and PFOM scores when benchmarked against the proxy ground truth.
